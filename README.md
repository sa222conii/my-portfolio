# Portfolio Website (sa222)

Web制作、Webアプリケーション開発のスキルと実績を掲載したポートフォリオサイトです。
Next.js, TypeScript, Tailwind CSSを使用して構築されています。

## 🛠 技術スタック (Tech Stack)

*   **Framework**: Next.js (App Router)
*   **Language**: TypeScript
*   **Styling**: Tailwind CSS
*   **Icons**: Lucide React
*   **Hosting**: Vercel

## 📂 掲載プロジェクト (Projects)

ポートフォリオ内で紹介している主な実績です。

### 1. SEO対策ツール (Web App)
OpenAI APIを活用したSEO分析ツール。
*   **主な機能**: キーワード・URL入力による競合分析・改善提案の自動生成、Google認証によるログイン、分析履歴の保存・一覧表示。
*   **技術**: Next.js, OpenAI API, Google Auth, Database
*   **URL**: [https://seo-ai-checker.vercel.app/](https://seo-ai-checker.vercel.app/)

### 2. 美容クリニック LP
30-40代女性をターゲットにした、ラグジュアリーで清潔感のあるランディングページ。
*   **特徴**: ターゲット層に響くデザイン選定、SEOを意識した構成により信頼感と集客力を最大化。
*   **技術**: HTML5, CSS, Design, Responsive

### 3. SNSマーケティング LP
動画制作やコンテンツ作成のクリエイティビティを強調した、モダンで動的なサービス紹介LP。
*   **特徴**: アニメーションを取り入れた視覚的インパクトのあるデザイン。
*   **技術**: HTML5, CSS, JavaScript, Animation

### 4. 税理士事務所サイト (WordPress)
既存の静的サイトを独自テーマ化し、CMSとして再構築。
*   **特徴**: クライアント自身で更新可能なお知らせ・ブログ機能の実装。
*   **技術**: WordPress, PHP, CMS, Theme Dev

### 5. セミナー資料作成 (スライド)
「伝わる」を最優先にしたセミナー用プレゼンテーション資料。
*   **特徴**: ターゲット層に合わせた論理的な構成と視認性の高いデザイン。
*   **ツール**: PowerPoint, Design, Presentation

## 🚀 開発環境のセットアップ (Getting Started)

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いて確認してください。
